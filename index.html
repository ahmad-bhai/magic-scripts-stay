Run on a site

Stay can't auto-inject into other apps the way a browser extension can. The reliable way on Android Chrome is a bookmarklet: a bookmark that runs your code on whatever page is open.

  1. Tap Copy link below.
  2. In Chrome, open ⋮ → Bookmarks → Add bookmark (any URL).
  3. Edit that bookmark's name to your script name, and paste the copied link as its URL.
  4. Open the target site, tap the address bar, type the bookmark's name, and select it to run the script.

How this app runs scripts

Android doesn't allow a plain web app to inject JavaScript into other websites or other apps in the background — that level of access is only available to real browser extensions (which is how the iOS Stay app works inside Safari).

This app gets you most of the way there: write and store your scripts here, then turn any of them into a bookmarklet — a one-tap bookmark that executes your code on whatever page is currently open in Chrome. It's the standard, working alternative on Android.
